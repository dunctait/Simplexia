# Simplexia Feature Backlog

## Phase 2 Candidates

- Add view modes for terrain color, heightmap, land mask, and biome mask, similar to procedural planet tools that expose multiple map diagnostics.
- Export PNG heightmaps, biome textures, JSON settings, and a bundled "export all" package.
- Add erosion and smoothing filters with before/after comparison.
- Add moisture and temperature maps so biomes can be classified from elevation, humidity, and latitude rather than fixed height bands.
- Add brush tools for raise, lower, flatten, paint biome, and blend biome edges.
- Add river/lake generation from downhill flow and rainfall.
- Add globe controls for pause rotation, reset camera, wireframe, and exaggerated elevation.
- Add globe-native equivalents for island masks, such as polar falloff, continental spread, and equatorial bias.
- Add a phase 2 "run" mode where saved generations become simulation inputs for deployable life forms.

## Research Notes

- MapNG emphasizes 3D previews and exporting heightmaps/textures for external tools.
- World Orogen uses multiple visualization modes and higher-order geology/climate controls.
- CotaMap combines generation with editing tools, erosion, smoothing, borders, and 2D/3D views.
- QuickMap groups workflows into generation, erosion, and brush tabs.
